# Security

## Supported usage

This repository is provided as open source reference software. It is not audited financial software.

## Rules

- Never commit `.env`
- Never reuse a high-value wallet
- Prefer a dedicated wallet with limited funds
- Start with `DRY_RUN=true`
- Treat every config change as a production change

## Reporting

If you find a serious security issue, do not publish the private key or sensitive account details in a public issue.
